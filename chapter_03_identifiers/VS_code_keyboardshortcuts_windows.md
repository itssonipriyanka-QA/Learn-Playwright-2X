# VS Code Keyboard Shortcuts - Windows

## General
| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + P` | Command Palette |
| `Ctrl + P` | Quick Open / Go to File |
| `Ctrl + ,` | Open Settings |
| `Ctrl + K Ctrl + S` | Open Keyboard Shortcuts |
| `Ctrl + Shift + X` | Open Extensions |
| `Ctrl + \`` | Open Terminal |
| `Ctrl + Shift + \`` | New Terminal |
