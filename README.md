# Learn Playwright 2X

A hands-on learning repository for Playwright automation, starting from JavaScript basics and progressing to end-to-end test automation.

## Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- [Playwright](https://playwright.dev/)

## Installation

```bash
npm install
npx playwright install
```

## Project Structure

```
Learn-Playwright-2X/
├── chapter_01_Basics/
│   ├── 01_Basics.js                        # JavaScript fundamentals (variables, arithmetic)
│   └── 01_command.js                       # Node.js process object (platform, arch, version, cwd)
├── chapter_03_identifiers/
│   ├── Identifier_Rules.js                 # Rules for naming identifiers in JavaScript
│   ├── identifier_Naming_convention.js     # Naming conventions (camelCase, PascalCase, etc.)
│   └── VS_code_keyboardshortcuts_windows.md # VS Code keyboard shortcuts for Windows
```

## Chapters

### Chapter 01 - JavaScript Basics

| File | Description |
|------|-------------|
| `01_Basics.js` | Introduction to JavaScript — variables, arithmetic operations, and console output |
| `01_command.js` | Node.js `process` object — platform, architecture, Node version, and current working directory |

### Chapter 03 - Identifiers

| File | Description |
|------|-------------|
| `Identifier_Rules.js` | Rules for valid JavaScript identifiers — allowed characters, reserved keywords |
| `identifier_Naming_convention.js` | Naming conventions — camelCase, PascalCase, snake_case, SCREAMING_SNAKE_CASE, Hungarian Notation |
| `VS_code_keyboardshortcuts_windows.md` | VS Code keyboard shortcuts reference for Windows |

## Running the Examples

```bash
node chapter_01_Basics/01_Basics.js
node chapter_01_Basics/01_command.js
node chapter_03_identifiers/Identifier_Rules.js
node chapter_03_identifiers/identifier_Naming_convention.js
```
